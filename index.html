安装Mysql

解压安装包

cd /home/app/software

mkdir -p /opt/mysql

tar -zxf mysql-5.7.23-1.el6.i686.rpm-bundle.tar -C /opt/mysql

安装各项依赖

cd /opt/mysql/

rpm -ivh mysql-community-common-5.7.25-1.el6.x86_64.rpm

rpm -ivh mysql-community-libs-5.7.25-1.el6.x86_64.rpm

rpm -ivh mysql-community-client-5.7.25-1.el6.x86_64.rpm

rpm -ivh mysql-community-server-5.7.25-1.el6.x86_64.rpm --nodeps

启动

service mysqld start

常用操作

重置初始密码

cat /var/log/mysqld.log | grep root@localhost

进入mysql命令行

mysql -u root -p

修改初始密码

注意：mysql命令行状态，输入命令以;结束，mysql对密码复杂度有校验标准（这个可以修改的），建议新密码包含英文大小写，数字，特殊字符

set password=password('Root@123');

进入mysql命令行状态

mysql -u root -p

输入密码，回车

查看mysql服务器状态

status;

授权远程连接

注意：防火墙启用3306

grant all privileges on *.* to 'root'@'%' identified by 'Root@123';

退出命令行状态

exit;

停止

/etc/init.d/mysqld stop

重启

service mysqld restart

查看mysql配置文件

cat /etc/my.cnf

启动常见报错

参见资料：

Can’t start server: can’t check PID filepath: No such file or directory

  1. 确认pid所在目录

    通过查看

  2. 建立目录并授权

    mkdir -p /var/run/mysqld/

    chmod -R 777 /var/run/mysqld/

  3. 启动mysql

    service mysqld start

部署Hexo+GitPage部署

Posted on 2020-04-18 Edited on 2020-05-04

部署Hexo+GitPage

Hexo安装

安装git

检查版本

git --version

安装 node

检查版本

node -v

安装 hexo

在本地建立一个文件夹（博客的目录），例如：E:\git\blog，进入这个文件夹，鼠标右键选择Git Bash Here

npm i -g hexo

hexo init

npm install

查看 hexo 版本

hexo version

Git建立博客仓库

仓库名必须为：用户名+github.io

Git设置SSH

查看配置的邮箱

git config --list

生成SSH公私钥

ssh-keygen -t rsa -C "邮箱地址"

会在C:\Users\${用户名}\.ssh下生成id_rsa和id_rsa.pub两个文件

配置公钥

登录github的个人账户，设置ssh公钥

测试是否授权生效

ssh git@github.com

出现You’ve successfully authenticated即成功

hexo 常用命令

下载主题

git clone https://github.com/theme-next/hexo-theme-next themes/next

修改_config.yml对应的主题 theme: next

新建Post

在blog目录下右键Git Bash here

hexo n “my fisrt blog”

在source\_posts文件夹下会出现my fisrt blog.md，在里面编写内容即可

删除Post

先删掉db.json这个文件，把对应的md文件也删掉，重新执行

hexo clean

hexo g

hexo d

清除数据

hexo clean

生成静态页面

hexo g

部署博客

在_config.yml的deploy下面配置

deploy:
type: git
repo: git@github.com:用户名/用户名.github.io.git
branch: master

然后执行hexo d
