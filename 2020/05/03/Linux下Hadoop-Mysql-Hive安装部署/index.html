Linux下Hadoop+Mysql+Hive安装部署

Posted on 2020-05-03 Edited on 2020-05-04

安装环境

CentOS Linux release 7.6.1810 (Core)

Linux master 3.10.0-957.el7.x86_64 #1 SMP Thu Nov 8 23:39:32 UTC 2018 x86_64 x86_64 x86_64 GNU/Linux

查看命令

cat /etc/redhat-release

uname -a

安装包准备

安装包/驱动 下载地址
jdk-7u80-linux-x64.tar.gz 下载
mysql-5.7.25-1.el6.x86_64.rpm-bundle.tar 建议直接用老师提供的那个
hadoop-2.7.7.tar.gz 下载
apache-hive-1.2.2-bin.tar.gz 下载
mysql-connector-java-5.1.6.jar 下载

关于mysql的下载，可以考虑使用wget

wget https://cdn.mysql.com//Downloads/MySQL-5.7/mysql-5.7.23-1.el6.i686.rpm-bundle.tar

目录安排

安装包存放目录：/home/app/software

安装目录：/opt

主机名配置

hostname master

echo master > /etc/hostname

Host配置

vi /etc/hosts

加入

按ESC键，输入:wq!保存hosts

重启后Host

/etc/init.d/network restart

部署JDK

解压

cd /home/app/software

tar -zxf jdk-7u80-linux-x64.tar.gz -C /opt

配置环境变量

vi /etc/profile

加入JAVA_HOME、JRE_HOME配置

按ESC键，输入:wq!保存profile

source /etc/profile

检查jdk环境变量

java -version

出现下图即配置成功

部署Hadoop

解压

cd /home/app/software

tar -zxf hadoop-2.7.7.tar.gz -C /opt

文件路径准备

hadoop运⾏行行时产⽣生⽂文件的存储⽬目录

mkdir -p /opt/hadoop-2.7.7/data/tmp

mkdir -p /opt/hadoop-2.7.7/data/var

mkdir -p /opt/hadoop-2.7.7/data/dfs/nn

mkdir -p /opt/hadoop-2.7.7/data/dfs/data

配置环境变量

vi /etc/profile

加入HADOOP_HOME配置

按ESC键，输入:wq!保存profile

source /etc/profile
